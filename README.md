> This project implements a **Product Management System** using **Python Data Structures**.
> It stores product details, tracks historical prices, and performs fast search & median price calculation using:
>
> ✅ Doubly Circular Linked List
> ✅ Binary Search Trees (Name-based & Price-based)
> ✅ Median calculation from linked-list data
>
> **Features**
>
> * Add new product & price history
> * Delete product
> * Find product by name
> * Display sorted products by price (lowest → highest / highest → lowest)
> * Show products in price range
> * Calculate median price of each product
>
> Perfect for learning **DSA implementation in Python**.
